Product Catalog KMP app

- Clean architecture
- Shared KMP modules for data, domain, presentation and app
- Lightweight MVI
- Ktor for client networking, consuming dummyjson.com API
- Room for persistence of favorite products
- Koin for DI
- Jetpack Compose for UI on Android
- Paging 3 for pagination
- Navigation 3 for navigation
- Some tests included in domain and presentation layers
